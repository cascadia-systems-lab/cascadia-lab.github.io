## How to Convert to PNG for LinkedIn

### Option 1: Online Converter (Easiest)
1. Go to: https://cloudconvert.com/svg-to-png
2. Upload the SVG file
3. Set dimensions:
   - Logo: 300 x 300 px
   - Banner: 1128 x 191 px
4. Download PNG

### Option 2: Using Browser
1. Open the SVG file in Chrome/Edge
2. Right-click → Inspect
3. Right-click on the `<svg>` element
4. Copy → Copy element
5. Paste into Figma, Canva, or Photopea
6. Export as PNG

### Option 3: Using Inkscape (Free Desktop App)
1. Download Inkscape: https://inkscape.org/
2. Open SVG file
3. File → Export PNG Image
4. Set dimensions and export

## LinkedIn Upload Instructions

### Upload Logo:
1. Go to your LinkedIn company page
2. Click the camera icon on the logo placeholder
3. Upload `linkedin-logo.png` (300 x 300 px)
4. Adjust positioning if needed
5. Click "Save"

### Upload Banner:
1. Click the camera icon on the banner area
2. Upload `linkedin-banner.png` (1128 x 191 px)
3. Preview and adjust
4. Click "Save"
